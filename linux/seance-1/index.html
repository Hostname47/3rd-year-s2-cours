Système d'exploitation (linux)

Notion générale sur le système d'exploitation ::

Définition l'ordinateur : est un ensemble de ressources matérielles (hardware) exemple processeur, mémoire, disque dur.... Un système d'exploitation (SE) : operating system en anglais (OS) est un ensemble de programme ( logiciel) qui permettent d'assurer la bonne gestion de l'ordinateur de ses périphérique.

-Il sert d'interface entre les ressources matérielles d'un ordinateur et les applications informatiques de l'utilisateur (software)

-Le système d'exploitation permet de géré : la mémoire, Les accès de périphérique, Le processeur, Contrôler la fiabilité du système La sécurité --> par feu
